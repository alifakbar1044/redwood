Ringkasan Desain

Fitur yang Digunakan

  • Font Awesome

    Ini digunakan untuk menambahkan ikon ke situs web

  • Javascript

    Ini digunakan dalam slider di formulir kontak untuk tujuan styling

  • Bootstrap

    Ini digunakan untuk menata beberapa elemen di halaman ini. Meskipun tidak digunakan di bagian lain dari situs web.
    Saya memutuskan untuk menulis semua CSS di situs web sendiri untuk belajar.

  • JQuery

    Ini digunakan pada preloader. Ini pada dasarnya menampilkan layar pemuatan sampai semua item di halaman telah dimuat sepenuhnya.
    Juga, ini digunakan di keranjang untuk menghapus item saat diminta oleh pengguna.

Pilihan Desain

Situs ini memiliki tema utama putih dan hitam dengan sedikit merah dan biru.
Situs ini adalah contoh situs web e-commerce yang hanya menjual sepatu untuk pria dan wanita.

  • Keranjang

    Keranjang dirancang agar pengguna dapat melihat total dan item yang dipilih berdampingan untuk memudahkan checkout

  • Navbar

    Navbar dirancang untuk memudahkan navigasi ke halaman web

  • Halaman Belanja

    Halaman yang menunjukkan item dirancang untuk melihat dengan mudah.
    Mereka menyesuaikan dengan ukuran layar dengan membungkus konten mereka.

  • Halaman Tentang

    Halaman ini dirancang untuk memberi tahu pengguna aspek-aspek terpenting dari organisasi (RD)
